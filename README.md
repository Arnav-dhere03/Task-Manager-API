# Enhanced Task Manager API

A RESTful Task Management API built using FastAPI.

## Features

- Create Tasks
- Update Tasks
- Delete Tasks
- View Tasks
- Task Priorities
- Task Status Tracking
- Pagination
- Filtering
- Sorting
- JWT Authentication
- User Management

## Tech Stack

- FastAPI
- SQLAlchemy
- SQLite
- Pydantic
- JWT Authentication
- Alembic
